Quick Start (5 Minutes)

Create Supabase Project

Go to supabase.com, create a new project, and copy your API keys.

Configure API Keys

Open js/supabase-config.js and replace the placeholder values with your actual Supabase URL and API key.

const SUPABASE_URL = 'your_supabase_url_here'; const SUPABASE_ANON_KEY = 'your_anon_key_here';

Setup Database

Copy the content from database-schema.sql and run it in your Supabase SQL Editor.

Create Storage Bucket

In Supabase Storage, create a new bucket named documents and set it as private.

Test the System

Open register.html in your browser, create an account, and start testing!

Important Notes

  • Read SUPABASE_SETUP_GUIDE.md for detailed setup instructions
  • The first registered user should be made admin via SQL query
  • All pages are fully responsive and production-ready
  • Email automation can be added using Resend or SendGrid

Before Going Live

  • Update Supabase redirect URLs with your production domain
  • Enable SSL/HTTPS on your hosting
  • Test all features thoroughly
  • Setup email automation for better user experience
  • Configure proper backup systems
